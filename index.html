Gamify WP is a WordPress Plugin that enables you to award points to your users for interacting with your self-hosted WordPress.org website.

Features

  • Award Points for Action Hooks added by other plugins

  • Assign Point Values for Clicking Links Via Shortcode

  • Limit Daily Points Awarded for Actions

  • Create Reward Levels and Badges

  • Display Detailed Stats to Users

  • Create Leaderboards

  • Track Site-wide Statistics with Graphs

  • No External API/Website Signup Needed
